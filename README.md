# TileSlicerJS

TileSlicerJS is a CLI tool / Node.js library to slice an image into XYZ tiles.

## Usage

```shell
$ git clone https://github.com/kamataryo/tile-slicer-js.git
$ cd tile-slicer-js
$ yarn
$ yarn start -h
$ yarn start sample.jpg ./docs/tiles --verbose --tiles-size=512
```
